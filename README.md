# Avance Madurez Digital

Proyecto front-end que muestra la pantalla inicial de la aplicación.

## Tecnologías

- React
- TypeScript
- Vite
- Tailwind CSS

## Desarrollo

Para ejecutar el proyecto localmente:

```bash
npm install
npm run dev
```

La aplicación estará disponible en `http://localhost:5173`
